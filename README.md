# Active-Directory-Home-Lab-Windows-Server-User-Identity-Management
VirtualBox home lab simulating a real IT environment using Windows Server and Active Directory. Focused on user, group, and access management, password resets, and basic Group Policy. Built to develop hands-on system administration and IT support skills aligned with junior sys admin and service desk roles.

## Overview
This project is a virtualised IT environment built using VirtualBox and Windows Server. It simulates a real-world business environment using Active Directory for user and identity management.

## Objectives
- Learn how Active Directory works in a domain environment  
- Practice user, group, and organisational unit (OU) management  
- Simulate IT support tasks such as password resets and account management  
- Build hands-on system administration skills for IT support and junior sysadmin roles  

## Technologies Used
- Windows Server (Active Directory Domain Services)
- VirtualBox
- Windows 10/11 Client VM

## Key Skills Practised
- User and group management  
- Password resets and account control  
- Organisational Unit (OU) structure  
- Basic Group Policy concepts  
- Troubleshooting and system configuration  

## What I Learned
- How domain environments work in real businesses  
- How IT administrators manage users and access  
- How support tickets translate into Active Directory tasks
